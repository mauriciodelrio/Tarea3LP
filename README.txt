1) Reglas para la Ejecución del Código:

- Entrar en el directorio donde se encuentra el archivo y escribir el siguiente comando:

	ant -f build.xml

- Donde 'build.xml' corresponde al archivo ANT. Luego, se debe ejecutar el archivo JAR, con el siguiente comando:

	java -jar t3lpfinal.jar
2)Consideraciones: 
-El juego fue desarrollado en netbeans, se recomienda importar el proyecto a netbeans, y ejecutarlo desde allí. 
-Los comentarios de las funciones se hicieron en la interfaz de plagiamon y de entrenador, debido a que estas son generales para todas aquellas clases que las implementan.
-Para el diseño de las clases graficas se utilizó el editor de netbeans.
